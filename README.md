# libnet

C++20 библиотека для работы с низкоуровневой сетевой адресацией, высокоточными типами данных и системными туннелями.

## 🏗 Архитектура проекта

Библиотека разделена на логические модули:

- **`core/`** — Базовые математические типы. Реализация `int128_t` и `uint128_t` для систем, где требуется 128-битная арифметика.
- **`network/`** — Сетевая логика L3. Работа с адресами IPv4, IPv6 и управление подсетями (IPv4Net/IPv6Net).
- **`dev/`** — Работа с системными устройствами. Абстракции над TUN/TAP интерфейсами для Linux.

## ✨ Особенности

- **Zero-dependency:** Основная логика не зависит от сторонних библиотек. (В процессе вндрение DPDK)
- **Modern C++:** Активное использование стандартов C++20.
- **Compile-time safety:** Строгая типизация и использование `noexcept` там, где это возможно.
- **Ready for Install:** Полная поддержка CMake install для интеграции в систему.
